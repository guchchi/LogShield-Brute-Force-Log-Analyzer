# LogShield Portfolio Summary

## Project Overview

LogShield is a Python-based defensive cybersecurity project that analyzes authentication logs and detects possible brute-force login behavior. It reads sample logs, runs detection rules, calculates severity, and generates an incident-style report.

## Why I Built It

I built this project to understand how authentication logs can reveal suspicious behavior. Log analysis is a fundamental skill in security operations, and this project helped me learn how repeated failed login patterns appear in log data.

## Cybersecurity Concepts Learned

- Brute-force detection — identifying rapid failed login attempts
- Log analysis — parsing and understanding authentication logs
- Authentication monitoring — tracking success/failure patterns
- Suspicious IP identification — flagging IPs with anomalous behavior
- Incident response basics — generating structured incident reports
- Severity scoring — classifying threats by risk level
- Defensive reporting — documenting findings and recommendations

## Technical Skills Demonstrated

- Python scripting
- Regular expression parsing
- Data aggregation and analysis
- Rule-based detection logic
- File I/O and report generation
- JSON data handling
