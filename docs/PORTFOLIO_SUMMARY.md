# LogShield Portfolio Summary

## Project Overview

LogShield is a Python-based defensive cybersecurity project that analyzes
authentication logs and detects possible brute-force login behavior.
It reads sample logs, runs detection rules, calculates severity, and
generates an incident-style report.

## Why I Built It

I built this project to understand how cybersecurity teams detect
suspicious login activity through logs. Log analysis is a fundamental
skill in security operations, and this project helped me learn how
attack patterns appear in authentication data.

## Cybersecurity Concepts Learned

- **Brute-force detection** — identifying rapid failed login attempts
- **Log analysis** — parsing and understanding authentication logs
- **Authentication monitoring** — tracking success/failure patterns
- **Suspicious IP identification** — flagging IPs with anomalous behavior
- **Incident response basics** — generating structured incident reports
- **SIEM/SOC-style thinking** — analyzing events, correlating data,
  and making security decisions
- **Severity scoring** — classifying threats by risk level
- **Defensive reporting** — documenting findings and recommendations

## Technical Skills Demonstrated

- Python scripting
- Regular expression parsing
- Data aggregation and analysis
- Rule-based detection logic
- File I/O and report generation
- JSON data handling

## Portfolio Statement

> This project helped me understand that cybersecurity is not only about
> finding vulnerabilities. It is also about monitoring systems, detecting
> suspicious behavior, and responding before damage happens.
